# ExMirror

学习用项目

### 演示站点

- 融合样式
- [MUI样式](https://ex.elysia.rip/)

融合样式，指`mini.css` + `MUI`，互相影响，问题有那么亿点点大

`MUI`样式由于组件数量过多，存在较大的性能问题

### 已经实现的功能

- [x] 画廊列表，画廊详情，画廊阅览(官方的多页阅览器+Ehviewer的解析页面实现的多页预览)

- [x] 画廊收藏

- [x] 只读(画廊标签，画廊评论)
